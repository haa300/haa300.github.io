HAA300 - A New Benchmark for Human-Centric Atomic Action Recognition

We propose a human-centric atomic action video dataset for ﬁne-grained action recognition. Instead of the existing and emerging atomic action datasets emphasizing data size, our dataset advocates to human-centric actions with an average of 77% detectable joints for the relevant human poses. It contains 300 human atomic action classes with 6K action labels in total. Each clip (taken from Youtube) lasts no more than 3 seconds. The collected human actions cover a broad range of classes including human-only actions such as waving hand, human-object interactions such as playing hulaloop, as well as human-human interactions such as hugging. We describe the dataset statistics, collection methodology, and compare quantitative performance for baseline neural network architectures trained and tested on human action classiﬁcation task using this dataset. We demonstrate ﬁne-grained action recognition by evaluating the AVA dataset with models pretrained with our dataset, and transfer learning results on UCF101 and HMDB51 after ﬁne-tuning our pretrained model.
